Tools / Postback Management

Postback
Management

Централизованный модуль управления постбеками — настройка, тестирование и мониторинг входящих и исходящих постбек-уведомлений для всех аффилиатских интеграций. Поддержка S2S-пикселей, макросов, фильтрации событий и логирования.

🔧
Module in development
Postback Management находится в активной разработке. Страница появится в одном из ближайших релизов.
Функциональность будет включать: S2S постбеки, шаблоны URL-макросов, тест-пуш, логи событий.
Target
v3.0
Planned Features
📡
S2S Postback Builder
Визуальный конструктор S2S-постбеков с поддержкой макросов: {click_id}, {amount}, {currency}, {event}, {brand}, {geo}. Шаблоны для популярных трекеров.
Planned
🧪
Test Push
Отправка тестовых постбеков с реальными или заглушечными параметрами. Проверка URL, статуса ответа и тела ответа в реальном времени.
Planned
📋
Event Log
Лог всех исходящих постбек-запросов с фильтрацией по аффилиату, событию, статусу, бренду и времени. Retry-механизм для неудачных запросов.
Planned
🔗
Affiliate Integration
Привязка постбек-шаблонов к аффилиатским аккаунтам. Управление правами и настройками на уровне компании, аффилиата и конкретного оффера.
Planned
⚙️
Event Mapping
Настройка событий: Registration, FTD, Deposit, Withdrawal, Chargeback. Фильтрация по GEO, бренду, типу трафика и минимальной сумме.
Planned
📊
Delivery Analytics
Метрики доставки: Success Rate, Avg Response Time, Failed%, Retry Rate. Алерты при деградации доставки у конкретного аффилиата.
Planned
