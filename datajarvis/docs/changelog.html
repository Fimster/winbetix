Changelog v2.3

История версий DataJarvis. Все изменения в спецификациях, прототипах и архитектурных решениях.

Latest: v2.3 March 2026

v2.3 — March 12, 2026

New

  • Affiliate Performance Report v1.0 — Notion ТЗ + HTML прототип. Drill-down L1→L2→L3, performance flags, side panel, 21 метрика
  • Architecture & Engineering Brief — Part 1 + Part 2. Data Pipeline, RLS матрица, кэширование, audit log, ETL idempotency
  • GitHub Project Structure — полная структура для разработки: index, reports/, docs/

Updated

  • Notion аудит — обновлены все страницы: Reports index, Vocabulary v1.4, Changelog
  • Navigation — рабочие ссылки между всеми прототипами

v2.2 — March 11, 2026

New

  • In/Out Report HTML Prototype v2 — Flat list, 4 вкладки, risk flags, hourly heatmap
  • In/Out Hover Tooltip — данные игрока при наведении на строку, следует за курсором
  • In/Out Side Panel — Player Journey, Today's Transactions, полный профиль

Updated

  • In/Out Report Notion ТЗ → v1.1 (flat list режим, hover tooltip спецификация)

v2.1 — March 10, 2026

New

  • DataJarvis TZ.docx — полное Word ТЗ с дизайном и таблицами
  • DataJarvis Presentation.pptx — 14 слайдов для stakeholders
  • Payment Report HTML Prototype — AR% heatmap, 3-уровневые метрики
  • Payment Report Notion ТЗ — страница #02

v2.0 — March 9, 2026

New

  • Churn Rate Report v2.0 — 7 KPI карточек со спарклайнами, группировка Early/Mid/Long
  • Строка ⊘ AVERAGE закреплена сверху когортной таблицы
  • Сворачиваемые группы колонок (Collapse/Expand)
  • Δ-колонки, drill-down по брендам, threshold highlight
  • Export: 4 листа (.xlsx) — Heatmap %, Absolute, Brand Comparison, KPI Summary

Breaking Changes

  • Churn Report ТЗ обновлено с v1.x до v2.0 — несовместимо с предыдущей структурой данных
