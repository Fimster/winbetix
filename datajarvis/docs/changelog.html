Changelog v2.4

История версий WinBetix. Все изменения в спецификациях, прототипах и архитектурных решениях.

Latest: v2.4 March 2026

v2.4 — March 13, 2026

Rebranding

  • DataJarvis → WinBetix — переименование платформы во всех HTML-прототипах, заголовках страниц и футерах
  • Логотип — заменён emoji-плейсхолдер на реальный logo.png во всех страницах

Shared Asset System

  • assets/styles.css — единый CSS для всех страниц: design tokens, reset, sidebar, topbar, badges, scrollbars. Дублирование CSS-переменных и reset-кода удалено из всех 6 report-страниц и index.html
  • assets/nav.js — общий sidebar-навигатор, инжектируемый в runtime. Правки навигации теперь в одном файле
  • assets/logo.png — логотип платформы, используемый всеми страницами

Refactoring

  • Все 6 report-страниц переведены на href="../assets/styles.css" и src="../assets/nav.js"
  • index.html переведён на href="assets/styles.css"
  • Удалено ~1100 строк дублирующегося CSS/JS кода из report-страниц
  • Обновлены все документационные страницы (architecture, design-system, overview, reports, vocabulary)

v2.3 — March 12, 2026

  • GitHub Project Structure — полная структура для разработки: index, reports/, docs/

Updated

  • Notion аудит — обновлены все страницы: Reports index, Vocabulary v1.4, Changelog
  • Navigation — рабочие ссылки между всеми прототипами
