Важно — NULL vs 0

NULL = данных нет (аффилиат не работал в этом GEO). 0 = данные есть, результат равен нулю. В UI: NULL → тире (—), 0 → "0" или "0%".

Key Formulas

NGR = GGR − Bonus Released NET PROFIT = NGR − Marketing Cost ROI % = NGR / Marketing Cost × 100% [NULL if Marketing Cost = 0] R2D % = FTDs / Regs × 100% [NULL if Regs = 0] eCPA = Marketing Cost / FTDs [NULL if FTDs = 0] AR% = Count Success / Count Attempts × 100% Retention% = Active(Mn) / Cohort Size × 100% M0 = 100% (always — base period)
⚠ Division by Zero

Все формулы с делением должны возвращать NULL при делителе = 0, а не Infinity или ошибку. Frontend отображает NULL как тире (—).
