01 — Overview

General Project Description

WinBetix — кастомный аналитический микросервис, заменяющий PowerBI. Централизованная платформа для всей аналитики брендов, аффилиатов и платёжных операций.

Active Development v2.3 March 2026

Project Goals

Основные цели проекта WinBetix:

  • Заменить PowerBI кастомным решением с полным контролем над данными и UI
  • Обеспечить real-time доступ к аналитике для всех отделов компании
  • Реализовать row-level security на уровне бренда, GEO и аффилиата
  • Создать единую платформу для 5+ типов отчётов с drill-down возможностями
  • Обеспечить экспорт данных в Excel (.xlsx) с форматированными листами

Alert System

Система уведомлений поддерживает два типа алертов:

  • Threshold Alert — срабатывает когда метрика пересекает заданный порог (например AR% < 60%)
  • Daily Digest — ежедневная сводка в 09:00 по brand timezone

Каналы: Email · Telegram · In-app уведомления

Дедупликация: cooldown 30 минут между одинаковыми алертами (configurable per rule)
