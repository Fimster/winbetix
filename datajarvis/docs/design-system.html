05 — Design System

WinBetix UI Design System

Полная документация дизайн-системы: CSS-переменные, типографика, компоненты. Единый источник правды для всех разработчиков и дизайнеров.

Dark Theme Only v1.0

Color Palette

Layout Structure

body (display: flex) ├── .sidebar (220px fixed) │ ├── .logo │ ├── .nav (.nav-section + .nav-item) │ └── .sidebar-bottom (.user-pill) └── .main (flex: 1, flex-direction: column) ├── .topbar (height: 52px, sticky) ├── .section-tabs ├── .filter-bar ├── .groupby-row ├── .kpi-wrap (.kpi-card ×N) └── .table-section (.table-wrap > table)

KPI Cards

Стандартная KPI-карточка содержит:

  • .kc-label — название метрики (DM Mono, 10px, uppercase)
  • .kc-val — основное значение (Syne, 22–28px, bold)
  • .kc-delta — изменение vs предыдущий период (зелёный/красный)
  • .mini-bars — спарклайн последних 7 значений (опционально)

Цветная полоска сверху карточки определяется через style="--card-color: var(--green)"

Table Conventions

  • Sticky первая колонка — всегда зафиксирована при горизонтальном скролле
  • th-group — цветные группы заголовков (Traffic, FTD, Deposits и т.д.)
  • col-sep — вертикальный разделитель между группами
  • Числа — всегда в DM Mono, выровнены по правому краю
  • NULL → "—" — тире, никогда не пустая ячейка
  • Hover строки — подсветка row + показ tooltip с деталями
