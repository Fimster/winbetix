About the Platform
What is WinBetix

WinBetix — внутренняя аналитическая платформа для мульти-брендовой iGaming-операции, заменяющая PowerBI.

Все отчёты работают с предварительно агрегированными данными — никаких real-time запросов к базе. SLA загрузки <3 сек для P95.

Права доступа управляются через Row-Level Security на уровне SQL — не на уровне API.

Architecture Principles
No real-time aggregations — pre-computed only
RLS enforced at SQL level, never API
Cursor-based pagination — no offset
UTC storage, per-brand timezone at query time
Idempotent ETL — UPSERT on conflict
Soft deletes only — immutable audit trail
